# Advizr Documentation

Public documentation for the Advizr AI transformation platform, built with [Nextra 4](https://nextra.site).

## Development

```bash
npm install
npm run dev
```

## Deployment

Deployed via Vercel. Push to `main` to trigger a production deploy.

## Structure

Content lives in `content/` as MDX files organized into 6 sections:

- **Platform** - Client portal guides
- **Services** - What Advizr offers
- **Architecture** - Technical reference
- **Academy** - Education content
- **Resources** - Guides, templates, integrations
- **Legal** - Terms and policies
