# Component Library

Reference page showing all custom components available in Advizr docs. Each component is globally registered and can be used in any MDX file without imports.

## Callout

Use callouts to highlight important information, warnings, or tips.

<Callout type="info">
  The Advizr API uses rate limiting. Free tier accounts are limited to 100 requests per minute.
</Callout>

<Callout type="warning" title="Breaking Change">
  The v1 authentication endpoint was deprecated in March 2026. Migrate to v2 before June 2026.
</Callout>

<Callout type="danger">
  Never share your API keys publicly or commit them to version control. Rotate any exposed keys immediately.
</Callout>

<Callout type="tip" title="Keyboard Shortcut">
  Press Cmd+K (or Ctrl+K on Windows) to open the command palette from anywhere in your dashboard.
</Callout>

<Callout type="note">
  This feature is available on Acceleration and Partnership plans only.
</Callout>

```mdx
<Callout type="warning" title="Breaking Change">
```

## Tabs

Use tabs to organize related content that users can switch between.

<Tabs>
  <Tab label="React">
    Install the Advizr SDK for React:

    ```bash
    npm install @advizr/react
    ```

    Then import and initialize:

    ```jsx
    import { AdvizrProvider } from '@advizr/react'
    ```
  </Tab>
  <Tab label="Python">
    Install the Advizr SDK for Python:

    ```bash
    pip install advizr
    ```

    Then import and initialize:

    ```python
    from advizr import Client
    client = Client(api_key="your-key")
    ```
  </Tab>
  <Tab label="CLI">
    Install the Advizr CLI globally:

    ```bash
    npm install -g @advizr/cli
    advizr init
    ```
  </Tab>
</Tabs>

```mdx
<Tabs>
  <Tab label="React">React content here</Tab>
  <Tab label="Python">Python content here</Tab>
</Tabs>
```

## Details

Use details for expandable content that is useful but not essential to the main flow.

<Details title="What browsers are supported?">
  Advizr supports all modern browsers including Chrome, Firefox, Safari, and Edge.
  Internet Explorer is not supported. We recommend keeping your browser updated
  to the latest version for the best experience.
</Details>

<Details title="How do I reset my password?" defaultOpen>
  Go to **Settings > Security > Change Password** in your dashboard, or use the
  "Forgot Password" link on the login page. You will receive a reset link via email
  within a few minutes.
</Details>
