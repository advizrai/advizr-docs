# System Architecture

How the Advizr platform is built — starting with the DOE framework that separates AI judgment from deterministic code.

We're rebuilding this section page by page with real commands, schemas, and diagrams pulled from the live systems. Sections reappear in the sidebar as they're completed; what's published here is verified, not aspirational.

## DOE at a glance

The entire system is built on three layers. Directives define the goal. Orchestration (AI) routes work and handles errors. Execution (Python) runs deterministic code with no probabilistic AI.
